Windows10 软件推荐

2018-10-22 • Windows • • • 783 • 2 min
他山之石可以攻玉

社交类

TIM

绿色版qq，功能简介启动迅速。现在qq用的越来越少了，每天上班开着纯粹是因为上学时养成了升级的习惯(ง •_•)ง

WeChat

代替了qq联络感情

钉钉

阿里出品，办公必备不用多说，除非你们公司没用钉钉~

浏览器

Chrome

谷歌浏览器，毫无争议，体验一级棒，丰富的插件，关于插件点击→宝藏

IE

系统自带浏览器，除非chrome不行了才会点你。(￣_￣|||)

UWP

说是实话不咋好用，感觉没人维护，还经常闪退。但必须推荐一下爱奇艺UWP、优酷UWP、搜狐UWP，看视频没广告！看视频没广告！看视频没广告！

日常工具

文件搜索

  • Listary 超级好用的文件搜索工具，按下快捷键，随时搜索
  • Everything 也是传说中的文档搜索神器！

笔记

  • 印象笔记 很好用的一款笔记，但是对支持markdown不怎么友好，需要装插件。
  • 有道云笔记 网易出品，单纯看中他的markdown，放弃了印象笔记转投有道云。

写作工具

  • typora 韩国的一款Markdown写作工具，免费的，简介好用！

文本编辑器

  • NotePad++替代了系统的文本编辑器.
  • Sublime Text 一款流行的代码编辑器软件，也是HTML和散文先进的文本编辑器。拥有强大的插件库
  • visual studio code 程序员必备软件，编写前端代码非常好用，也支持markdown。

快速启动

  • Wox 也是一个软件启动神器，但是，上次体验之后被我打入冷宫了，不知道现在怎么样，总之，值得体验！

快速预览文件

  • QuickLook 可实现预览word，txt，pdf，rar等等文件

命令工具

  • CmderWindows环境下非常简洁美观易用的cmd替代者，它支持了大部分的Linux命令。支持ssh连接linux，使用起来非常方便。

视频播放器

  • 射手影音播放器 小巧开源，启动迅速。

下载工具

  • 迅雷5.8 雷5.8稳定版的去广告绿色精简版本，无广告，下载速度还挺快。

安卓模拟器

  • 夜游神模拟器 电脑上玩手机游戏必备软件

桌面整理

  • 猎豹轻桌面 小巧，消耗内存少，桌面图标分类整理
  • 火萤酱 本地搜索、网络搜索和小程序。

远程控制

  • TeamView 最好用的远程控制软件

解压软件

  • 7zip 著名解压软件，也是免费

电子书

  • calibre 用来推送kindle再好不过了。还能将其他格式的电子书转换

邮箱

  • Foxmail 除了接收邮件，还有RSS订阅功能！

思维导图

  • Xmind 思维导图，头脑风暴必备。

广告屏蔽

  • ADsafe 净网大师是国内首款免费专业净网软件
  • 阿呆喵 ADM一直致力于广告拦截的开发，并坚持免费提供使用

GIF录制

  • LicCam 用来录制 GIF 动画的

定时关机

  • 定时关机助理 其实可以用系统内置的任务,这个小工具更易用
